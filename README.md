## Project Structure
```
├── Pokemon.h
|   ├── travel.c
|   ├── buy.c
|   ├── enemyAttack.c
|   └── fight.c
├── Pokemon.c
|   └── Pokemon_list.txt
```

- Pokemon.h
게임 진행에 필요한 모듈들의 헤더 파일
  
  - travel.c : 여행을 떠나는 함수
  - buy.c : 상점가는 함수
  - enemyAttack.c : 적이 공격하는 함수
  - fight.c : 적과 싸우는 함수

- Pokemon.c
게임 소스 코드
  - Pokemon_list.txt : 포켓몬 목록 txt
